# 复合选择器

## 交集选择器

作用：选中同时复合多个条件的元素
语法：选择器1选择器2选择器3选择器n{}
注意点：交集选择器中如果有元素选择器，必须使用元素选择器开头

```CSS
三个类同时满足
.a.b.c{
    color: blue
}

选择div下的red类
div.red{
    font-size: 30px;
}
```

## 并集选择器(,)

作用：同时选择多个选择器对应的元素
语法：选择器1,选择器2,选择器3,选择器n{}

```css
<h1>标题</h1>

<span>哈哈</span>

h1, span{
    color: green
}
```

同时选中h1和span
